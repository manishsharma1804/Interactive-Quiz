# Interactive Quiz

Welcome to the Interactive Quiz project by [MANISH SHARMA](https://www.instagram.com/me.manish18/)!

This project is a simple web-based quiz application designed to provide an interactive and engaging quiz experience.

## Features

- **Timed Questions**: Each question has a time limit of 15 seconds.
- **Scoring**: Points are awarded based on correct answers.
- **Interactive Interface**: Easy navigation with start, next, replay, and quit buttons.
- **Responsive Design**: Supports various screen sizes for optimal viewing.
- **Rules**: Displays quiz rules before starting.
